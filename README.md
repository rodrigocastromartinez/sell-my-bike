## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
    npx expo start
   ```

## Run tests

For unit tests, from the project folder, run:

```bash
npm run test:unit
```

For e2e tests, from the maestro folder, run:

```bash
maestro test home.yaml
```
